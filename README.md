📊 Predictive Analytics Project
Sales Forecasting Using Python
🧾 Overview

This project focuses on analyzing historical sales data to understand sales performance and predict future sales trends using machine learning techniques. The process follows a complete data science workflow including data preprocessing, exploratory data analysis (EDA), visualization, and forecasting.

The main objective is to identify sales patterns and build a predictive model to forecast future sales.

📁 Dataset Information
Dataset: Sales Data (CSV file)
Key Features:
Order Date
Sales
Region
Other business-related attributes
🎯 Objectives
Analyze historical sales data
Understand overall sales performance
Identify trends over time
Compare sales across regions
Build a forecasting model for future sales prediction
Visualize insights for better business decisions
🔧 Data Preprocessing
Loaded dataset using pandas
Converted Order Date into datetime format
Removed duplicate records
Handled missing values
Sorted data in chronological order
📊 Exploratory Data Analysis (EDA)
Displayed dataset structure and summary
Checked data types and missing values
Calculated key metrics:
Total Sales
Average Sales
Maximum Sales
Minimum Sales
📈 Data Visualization
1️⃣ Sales Over Time (Line Chart)
Shows how sales change over time
Helps identify increasing or decreasing trends
2️⃣ Sales by Region (Bar Chart)
Compares total sales across different regions
Helps identify high-performing regions
🤖 Machine Learning Model
📌 Model Used: Linear Regression
Created a time-based feature called Days
Used Days as input (X) and Sales as output (Y)
Trained Linear Regression model using sklearn
Predicted next 30 days of sales
📉 Forecasting Result
Actual sales trend was plotted
Predicted future sales were added as a dashed line
Helps visualize future business performance
